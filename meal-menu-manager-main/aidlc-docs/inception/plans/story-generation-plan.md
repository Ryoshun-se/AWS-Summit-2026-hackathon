## 確定した生成方針

| 項目 | 決定内容 |
|---|---|
| **ペルソナ** | 2 ペルソナ（子育て中の親 + 共働き夫婦） |
| **分解アプローチ** | ユーザージャーニー基準（週次サイクルの流れに沿って整理） |
| **粒度** | 粗め（エピックレベル、5〜10 ストーリー） |
| **受け入れ基準** | 標準（3〜5 条件、Given/When/Then 形式） |
| **コンセプト表現** | 通常形式（「選択行為ゼロ」は背景として記載） |
| **優先度** | MoSCoW 法（Must/Should/Could/Won't） |
