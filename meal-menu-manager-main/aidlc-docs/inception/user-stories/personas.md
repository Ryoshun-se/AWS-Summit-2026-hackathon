# ユーザーペルソナ

> **アプリコンセプト**: 食事に関する選択行為をゼロにする — 「人をダメにする」アプリ  
> **共通ニーズ**: ライフスタイルや健康目標は異なっても、「食事の意思決定を減らしたい」という共通ニーズを持つ

---

## Persona 1: 子育て中の親 — 田中 さくら（38歳）
**ライフスタイルモード**: ファミリーモード

### プロフィール
- **職業**: パートタイム勤務（週 3 日）
- **家族構成**: 夫、子供 2 人（5 歳・8 歳）
- **技術リテラシー**: 普通

### 課題・ペインポイント
- 毎日「今日の夕食何にしよう」と悩む時間を 15〜30 分消費している
- 子供が食べられるか、先週食べたかを毎回確認するのが大変
- 旬の食材や特売品を活かした献立を考える余裕がない

### アプリへの期待
- 何もしなくても週の献立が自動で決まっていてほしい
- 子供 OK の料理だけを提案してほしい
- 買い物リストがそのまま使えるレベルで出てきてほしい

### 関連ユーザーストーリー
- US-01, US-02, US-03, US-04, US-05, US-06, US-07

- **技術リテラシー**: 高い

### アプリへの期待

## Persona 3: ボディメイク実践者 — 山田 拓也（28歳）
**ライフスタイルモード**: ボディメイクモード

### プロフィール
- **職業**: 会社員（週 4 回ジム通い）
- **家族構成**: 一人暮らし
- **技術リテラシー**: 高い

### 課題・ペインポイント
- タンパク質・脂質・炭水化物（PFC）のバランスを保ちながらバリエーションを出すのが難しい
- 毎回同じような食事（鶏胸肉・ブロッコリー・玄米）で飽きてしまう
- 筋トレ後のリカバリーに何を食べればいいか分からず、いつも同じものを食べてしまう
- PFC を計算しながら献立を考えるのに時間がかかる

### アプリへの期待
- PFC 目標値を設定すれば、自動でバランスの取れた献立を提案してほしい
- 同じ食材でも調理法を変えてバリエーションを出してほしい
- 筋トレ後のリカバリーを考慮した食事を提案してほしい

### 関連ユーザーストーリー
- US-01, US-02, US-03, US-04, US-06, US-07, US-09

## Persona 4: 生活習慣病のシニア — 佐藤 正雄（67歳）
**ライフスタイルモード**: ヘルスケアモード

### プロフィール
- **職業**: 定年退職後（妻と 2 人暮らし）
- **健康状態**: 高血圧・軽度の糖尿病と診断
- **技術リテラシー**: 低め（スマホは使えるが操作は簡単なものに限る）

### 課題・ペインポイント
- 医師から塩分 6g/日以下・脂質制限を指示されているが、制約の中で献立を考えるのが大変
- 健康管理の本を参考に献立を考えているが、同じメニューになりがち
- 妻と 2 人分の食事を毎日考えるのが負担になっている
- 制約を守りながら美味しい食事を楽しみたいが、アイデアが尽きる

### アプリへの期待
- 塩分・脂質の制約値を設定すれば、自動で守ってくれる献立を提案してほしい
- 制約の中でも飽きない多様な献立を提案してほしい
- 操作がシンプルで分かりやすいアプリにしてほしい

### 関連ユーザーストーリー
- US-01, US-02, US-03, US-04, US-06, US-07, US-10

## Persona 5: 一人暮らしの学生・社会人 — 中村 あおい（22歳）
**ライフスタイルモード**: ソロモード

### プロフィール
- **職業**: 大学生（アルバイト掛け持ち）
- **家族構成**: 一人暮らし
- **技術リテラシー**: 高い（スマホネイティブ）

### 課題・ペインポイント
- 外食が多く栄養バランスが乱れがち。自炊する日は健康的なものを作りたいが思いつかない
- 1 人分の調理だと食材が余って廃棄が多い
- 食費予算が月 2〜3 万円と限られており、安い食材に偏りがち
- 決まったメニュー（パスタ・炒め物）ばかりになってバリエーションを増やしたい

### アプリへの期待
- 食材が余らない 1 人分の献立を提案してほしい
- 食費予算内に収まる食材で献立を作ってほしい
- 外食した日の栄養バランスを補完する自炊メニューを提案してほしい

### 関連ユーザーストーリー
- US-01, US-02, US-03, US-04, US-06, US-07, US-11

## ペルソナ共通項の抽象化

全ペルソナに共通する本質的なニーズ：

| 共通ニーズ | 説明 |
|---|---|
| **選択疲れからの解放** | 食事の意思決定を減らし、脳のリソースを他に使いたい |
| **バリエーションの確保** | 同じ食事の繰り返しを避け、飽きない献立を自動で提案してほしい |
| **制約の自動遵守** | 健康目標・医療制約・予算・食材廃棄など、各自の制約を AI に任せたい |
| **履歴学習** | 過去に食べたものを記憶して重複を避けてほしい |
| **買い物の効率化** | 献立から買い物リストを自動生成してほしい |

これらの共通ニーズを満たす共通アーキテクチャの上に、ライフスタイルモードによるカスタマイズ層を設けることで、全ペルソナに対応する。
